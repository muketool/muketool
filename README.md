**离线安装的两种方法：**

> **方法一：**
>
> ① 下载上方插件的离线安装版（扩展名为.crx的文件）。
>
> ② 点击谷歌浏览器右上角的自定义及控制按钮，在下拉框中选择工具选项，然后点击扩展程序来启动浏览器的扩展管理器页面。
>
> ③ 在打开的浏览器的扩展管理器中用户可以看到一些已经安装程序的插件，或者一个插件也没有。在安装本地插件的之前勾选开发者模式，然后把我们的插件直接拖到下方的插件区域，就自动安装了。

> **方法二：**
>
> ① 由于一些特殊原因，极个别插件在使用上述的安装方法的时候，浏览器会提示“只能通过网上应用商店安装该程序”。
>
> ② 把下载后的.crx扩展名的离线插件的文件扩展名改成.zip或者.rar。
>
> ③ 右键点击该文件，并使用压缩软件（如winrar、好压、360压缩等）对该压缩文件进行解压，并保存到系统的一个任意文件夹下（安装成功后该文件夹不能删除）。
>
> ④ 解压成功以后，该插件就会以文件夹的形式存在于操作系统的某一个目录下面，打开浏览器的扩展程序管理界面。在勾选开发者模式选项以后，在该页面就会出现 “加载正在开发的扩展程序” 等按钮，点击 “加载正在开发的扩展程序” 按钮，并选择刚刚解压的插件文件夹的位置。
>
> ⑤ 这时候如果不出意外，插件就会成功加载到浏览器中。
>
> ⑥ 基于这种模式安装的插件会因为用户启用了开发者模式而遭到浏览器的警告，用户可以选择忽略浏览器的警告。

### 2. 安装脚本
